🌐 EDBP (Easy Discord bot Plugin) のご紹介

🚀 EDBPとは？

EDBP (Easy Discord bot Plugin) は、あなたのDiscordボットをさらに強力にするための拡張機能システムです。

EDBPは、EDBB (Easy Discord bot Builder) に標準搭載されている強力な機能の一つで、ユーザーはコーディングの知識がなくても、ボットに多彩な機能を手軽に追加・拡張できます。

これにより、開発者はもちろん、初めてボットを作成する方でも、自分の理想とするDiscordボットを簡単に実現できます！

🤖 プラグインの導入方法

EDBPでは、プラグインを非常に簡単かつ安全にボットへ組み込むことができます。導入方法は主に2通りあります。

1. 公式SHOPからの導入 (推奨)

EDBP内のSHOP機能を利用すれば、必要なプラグインを自動でインストールできます。面倒なファイル操作は一切不要です。

2. 手動での導入

SHOPにない独自のプラグインや、特定のファイル形式で共有されたプラグインをインストールする際にご利用いただけます。

✨ プラグイン情報の取得（仕組みの説明）

あなたのウェブページでプラグインのリストを表示するために、EDBPはGitHubのタグを利用して必要な情報を取得します。

この仕組みにより、開発者はGitHub上でプラグインを管理し、ユーザーはウェブサイトを通じて最新の情報を確認できます。

📥 取得されるデータ項目

EDBPがGitHubリポジトリの特定のタグ（edbp-plugin）から取得し、一覧表示に利用する主なデータは以下の通りです。

データ項目 説明
名前 プラグインの名称
作成日 プラグインが公開された日付
作成者 プラグインの開発者名
バージョン プラグインの現在のバージョン情報
星の数 ユーザーからの評価（人気度を示す）
説明 プラグインの機能の概要
